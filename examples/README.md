# 示例项目

此目录存放论文项目示例。

## 目录结构

每个示例项目应包含以下结构：

```
<项目名>_<日期>/
├── README.md           # 项目说明
├── 格式规范.md          # 格式规范文档
├── 资料汇编.md          # 资料整理
├── 参考文献.md          # 参考文献列表
├── 论文大纲.md          # 论文结构大纲
├── content/            # 各章节内容
│   ├── 00_摘要.md
│   ├── 01_引言.md
│   └── ...
└── output/             # HTML 输出
    └── 论文.html
```

## 添加示例

完成一个论文项目后，可将其复制到此目录作为示例供参考。
