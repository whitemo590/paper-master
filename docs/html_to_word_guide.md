# HTML 转 Word 指南

本文档介绍如何将 AI 生成的 HTML 文件完美复制到 Word 中。

## 🤔 为什么使用 HTML？

### 问题

直接把 AI 生成的文本粘贴到 Word：

- ❌ 本质上是粘贴"纯文本"或"Markdown"
- ❌ Word 对这些格式的兼容性不完美
- ❌ 粘贴进去的内容排版完全对不上

### 解决方案

Word 和 HTML 的底层富文本结构具有**极高通用性**。

- ✅ 让 AI 生成"带有排版样式的 HTML 代码"
- ✅ 在浏览器中打开 HTML 文件
- ✅ 全选复制 → 直接保留所有排版样式粘贴到 Word

---

## 📋 操作步骤

### 步骤 1：找到 HTML 文件

论文完成后，HTML 文件位于：

```
projects/<项目名>/output/论文.html
```

### 步骤 2：在浏览器中打开

- 双击 HTML 文件，自动在默认浏览器中打开
- 或右键 → 打开方式 → 选择浏览器（推荐 Chrome/Edge）

### 步骤 3：全选复制

```
Windows: Ctrl + A（全选） → Ctrl + C（复制）
Mac: Cmd + A（全选） → Cmd + C（复制）
```

### 步骤 4：粘贴到 Word

1. 打开 Word，新建文档
2. 粘贴：`Ctrl + V`（Windows）或 `Cmd + V`（Mac）
3. 格式将完整保留！

---

## 🔧 粘贴后调整

### 需要手动处理的内容

| 内容 | 操作 |
|------|------|
| **页边距** | 布局 → 页边距 → 适中/自定义 |
| **页眉页脚** | 插入 → 页眉/页脚 |
| **页码** | 插入 → 页码 |
| **目录** | 引用 → 目录 → 自动目录 |
| **封面** | 自行添加封面页 |

### 格式微调

如果某些格式不正确：

1. 选中对应文本
2. 在"开始"选项卡中调整字体、字号、行距等
3. 或使用"格式刷"复制其他正确格式

---

## 💡 技巧

### 选择性粘贴

如果不想要某些样式，可以使用"选择性粘贴"：

1. `Ctrl + Alt + V`（Windows）
2. 选择"纯文本"或"保留源格式"

### 字体缺失问题

如果 Word 中字体显示不正确：

1. 全选文档（Ctrl + A）
2. 修改字体：
   - 中文：宋体
   - 英文：Times New Roman

### 行距调整

1. 全选文档
2. 右键 → 段落
3. 设置行距为 1.5 倍或固定值

---

## ⚠️ 注意事项

1. **推荐浏览器**：Chrome / Edge（兼容性最好）
2. **不要直接打开源代码**：必须在浏览器中渲染后复制
3. **检查特殊字符**：公式、特殊符号可能需要手动调整
4. **保存为 .docx**：粘贴后记得保存

---

## 🔍 常见问题

### Q: 粘贴后格式全乱了？

**A**: 可能原因：
1. 没有在浏览器中打开，而是复制了源代码
2. Word 版本过低
3. 浏览器渲染问题

**解决**：尝试使用 Chrome 浏览器重新操作

### Q: 表格变形了？

**A**: Word 对表格宽度的处理可能不同，粘贴后手动调整表格宽度。

### Q: 中文字体变成方框？

**A**: 字体缺失，全选后手动设置为"宋体"。
